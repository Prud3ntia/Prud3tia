# CompTIA A+ 220-1201
# Lesson 1.1 – Laptop Hardware

## Lesson Objective

Understand the hardware unique to laptops, common upgrade options, battery technology, storage devices, wireless technologies, and biometric authentication.

---

# Laptop Repair

- Laptop repair is more challenging than desktop repair.
- Requires precise troubleshooting due to the compact design.
- Some laptop components are modular and easy to replace.
- Other components are built into the system and are not designed to be replaced by the user.

---

# Laptop Batteries

## Common Battery Types

- Lithium-Ion (Li-ion)
- Lithium Polymer (LiPo)

### Important Facts

- Modern batteries have **no memory effect**.
- Charging the battery does not reduce its overall capacity.
- Battery form factors vary between manufacturers and laptop models.
- Battery technology continues to evolve.

### Modular Batteries

- Usually removable with a release button.
- Easier to replace than internal batteries.

---

# Laptop Keyboard

- The keyboard is one of the most frequently used laptop components.
- It can often be replaced by removing a few screws and disconnecting a ribbon cable.
- Some laptop models require additional disassembly.
- Individual keycap replacement can be delicate and should be done carefully.

### Troubleshooting Tip

Use an external USB keyboard to determine whether the built-in keyboard is faulty.

---

# Laptop Memory (RAM)

## SO-DIMM

**SO-DIMM** stands for **Small Outline Dual Inline Memory Module**.

### Facts

- Used in laptops and other mobile devices.
- Smaller than desktop DIMMs.
- Often easy to install or replace.
- Some laptops have RAM soldered directly to the motherboard.
- Soldered RAM cannot be upgraded and requires motherboard replacement if it fails.

---

# Storage Devices

## Hard Disk Drive (HDD)

- Traditional spinning magnetic disks.
- Common laptop size: **2.5-inch**
- Desktop HDDs commonly use **3.5-inch** drives.

---

## Solid State Drive (SSD)

- No moving parts.
- Faster access times.
- Lower latency.
- Silent operation.
- Uses less power.

---

## M.2 SSD

### Advantages

- Smaller form factor.
- No SATA power cable required.
- No SATA data cable required.
- Easy to install and replace.
- Usually secured with a single screw.

---

# Replacing Laptop Storage

Depending on the laptop:

- Remove a rear access cover.
- Open the entire laptop.
- Remove the storage drive.
- Install the replacement drive.

Many modern laptops are highly modular.

---

# Upgrading from HDD to SSD

One of the best upgrades you can make.

Benefits:

- Much faster boot times.
- Faster application loading.
- Better overall system performance.
- Can make an older laptop feel like a new computer.

### Option 1 – Fresh Installation

- Install a new operating system on the SSD.
- Transfer user files.
- Reinstall applications.

Pros:
- Clean installation.

Cons:
- Time consuming.

---

### Option 2 – Clone/Image the Drive

- Copies everything from the HDD to the SSD.
- No operating system installation required.

Requires:

- Disk imaging or cloning software.

Methods:

- Create an image file.
- Clone directly from one drive to another.

# Wireless Technologies

## Wi-Fi (802.11)

- Wireless Local Area Network (LAN).
- High-speed wireless networking.
- Used for internet access.

---

## Bluetooth

- Personal Area Network (PAN).
- Short-range wireless communication.
- Used to connect nearby peripherals.

Examples:

- Mouse
- Keyboard
- Headphones
- Speakers

---

# Laptop Wi-Fi Antennas

Most laptops contain multiple antennas.

Examples:

- Main Wi-Fi antenna
- Auxiliary Wi-Fi antenna
- Bluetooth antenna

These antennas are usually routed around the laptop display.

---

# Biometrics

Used to unlock or sign into a device using:

- Fingerprint recognition
- Facial recognition

### Requirements

- Compatible hardware.
- Operating system configuration.

Benefits:

- Secure.
- Convenient.
- Difficult to duplicate.

---

# Near Field Communication (NFC)

### Uses

- Mobile payments.
- Authentication.
- Small data transfers.

### Range

Approximately **3 centimeters or less**.

Common Uses

- Hospitals
- Warehouses
- Manufacturing facilities

Allows authentication without typing a password.

---

# Webcam & Microphone

Most laptops include:

- Built-in webcam.
- Built-in microphone.

Common Uses

- Video conferencing.
- Online meetings.
- Video recording.

External USB or analog microphones often provide better audio quality.

# Key Takeaways

- Laptop repair is more difficult than desktop repair due to compact components.
- SO-DIMM is the standard RAM used in laptops.
- SSDs provide a significant performance improvement over HDDs.
- M.2 SSDs are compact and easy to install.
- Wi-Fi provides wireless LAN connectivity, while Bluetooth connects nearby peripherals.
- NFC enables very short-range authentication and payments.
- Biometrics improve security through fingerprint or facial recognition.
